# Shopping Cart Management System in Java

This is a console-based Shopping Cart Management System built in Java.  
It demonstrates object-oriented programming concepts such as classes, objects, encapsulation, inheritance, method overriding, exception handling, and the singleton pattern.

## Features

- Add products to cart
- Add discounted products
- Remove products from cart
- View receipt
- Checkout system
- Total price calculation
- Custom exception handling

## Concepts Used

- Java classes and objects
- Encapsulation
- Inheritance
- Method overriding
- ArrayList
- Scanner input
- Custom exceptions
- Singleton pattern

## How to Run

1. Open the project in NetBeans, IntelliJ IDEA, Eclipse, or VS Code.
2. Compile the `Shoppingcart.java` file.
3. Run the `Shoppingcart` class.
4. Use the menu options to add products, remove products, view receipt, and checkout.

## Project Status

This is a beginner-level Java OOP project created for learning and portfolio building.
